# Diwali Sales Data Analysis

This repository contains a Python-based Jupyter Notebook that analyzes Diwali sales data to uncover key purchasing patterns, customer demographics, and high-performing product categories.

## 🚀 Key Insights & Objectives
* **Customer Demographics:** Analyzed sales distribution across gender, age groups, and marital status to identify the target consumer base.
* **Geographical Analysis:** Identified the top-performing states and zones contributing the highest revenue and order volume.
* **Product Performance:** Evaluated the most sold product categories (such as Clothing, Electronics, and Food) to optimize inventory planning.

## 🛠️ Technologies Used
* **Python**
* **Jupyter Notebook**
* **Pandas / NumPy** (Data Cleaning & Manipulation)
* **Matplotlib / Seaborn** (Data Visualization)
